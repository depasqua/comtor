Spell Check Display pre-/post-analysis content

This module compares words found in comments against an English word dictionary. The COMTOR word dictionary contains more than 97,000 entries. The analysis includes the list of known Java API classes (JDK), HTML tags and attributes, Java keywords, and user-defined symbols from the submitted code (class, method, constructor, field names). COMTOR uses JavaDoc, as such JavaDoc does not provide access to local variables within methods and blocks. Thus, the word list is not exhaustive.
