About COMTOR

COMTOR is a web-based service designed to evaluates various aspects of Java source code commenting. To read more the feature set that COMTOR provides, click on Features We Measure menu item.

To use COMTOR you will Log In with your email address, sign up for the appropriate class section, and submit your code for evaluation.

Creating an Account and Logging In

If this is your first time to the site you will need to create an account. To continue, click on Create An Account menu item.

If you already have an account, enter your registered email address and the password the system mailed to you, and you will be taken to the welcome screen. To recover your password, click on the Password Recovery menu item.
